Exécution du code :

Cette partie représente l'exécution du code, comment fonctionne le programme.

Une fois le code lancé, on obtient cela :

C'est une sorte de menu, avec plusieurs questions qui arrivent au fur et à mesure.

Si vous voulez jouer une partie personnalisé, il suffit de mettre n et ensuite le programme vous demande le nombre de ligne et de colonne ainsi que le nombre d'alignement :

On peut jouer à deux personnes ou contre une intélligence artificelle. Pour pouvoir placer un jeton, il suffit de mettre un chiffre entre 1 et 7. Par exemple, vous voulez mettre un jeton dans la colonne 6, taper 6 dans "Entrez le numero de colonne (1000 pour passer son tour)" A noter qu'on ne peut pas passer son tour dans une partie classique, c'est uniquement dans la partie personnalisé.
